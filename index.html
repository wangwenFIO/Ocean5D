介绍

Ocean5D程序是一款对海洋水文环境数据进行多维可视化显示的应用软件。Ocean5D基于OpenGL和OSG（Open Scene Graph）可视化渲染引擎开发实现，部分模块采用GLSL和CUDA等GPU并行计算和渲染技术，以提高大规模海洋环境数据的建模和渲染效率。Ocean5D目前只支持NetCDF（Network Common Data Form）格式数据的加载。Ocean5D正在不断完善和改进中。

Ocean5D可自由获取，仅能用于海洋科学研究以及相关学习、教育活动，不得用于商业用途。

特点

Ocean5D程序有以下特点:

  • 支持NetCDF（Network Common Data Form）格式数据；
  • 在三维立体环境下，对海洋水文环境数据进行多要素、多视角、交互式动态展示；
  • 可自由加载一个或多个深度面、纬度面、经度面以及自定制切面对海洋水文环境数据进行动态展示；
  • 可同步显示指定位置和要素的时间序列曲线和深度变化曲线；
  • 采用体绘制的方式对海洋水文环境数据进行交互式动态展示；
  • 采用CUDA（Computing Unified Device Architecture）技术实现部分模块的并行计算；
  • 应用GLSL着色器编程技术实现海洋环境数据的快速计算和渲染。
